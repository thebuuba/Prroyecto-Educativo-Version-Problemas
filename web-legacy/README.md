# AulaBase Legacy JS

Este subproyecto sirve la app JavaScript heredada que vive en `public/legacy`.

## Comandos

- `npm run dev`: levanta el servidor local con Vite.
- `npm run build`: valida y empaqueta la app legacy.
- `npm run lint`: revisa el JavaScript del proyecto.

## Notas

- El punto de entrada principal está configurado en `vite.config.js`.
- La app legacy funciona sin JSX ni componentes de framework.
