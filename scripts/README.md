# Scripts

Scripts activos del proyecto:

- `assemble-index-html.sh`: ensambla `index.html` desde los fragmentos fuente.
- `assemble-all.sh`: ejecuta el ensamblado fuente actual.

Para preparar una publicación estática usa `npm run prepare:dist`.

Los scripts antiguos de migración a español, reescritura masiva y hosting legacy fueron eliminados porque usaban rutas absolutas o artefactos generados que ya no corresponden a la arquitectura actual.
